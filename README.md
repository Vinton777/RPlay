## üé≠ Commands

| Command | Description |
|------|------------|
| `/me <text>` | Local roleplay action |
| `/gme <text>` | Global roleplay action |
| `/do <text>` | Roleplay description |
| `/roll [max]` | Random roll |
| `/try <text>` | Success / fail action |
| `/rplay reload` | Reload plugin config |
